### 變更說明

修復報表頁面 TypeScript 錯誤。recharts Tooltip 的 Formatter 類型與自定義 formatter 函數不兼容，導致 TypeScript 編譯失敗。

**問題**：
- `pipelineTooltipFormatter` 返回類型與 recharts `Formatter` 類型不匹配
- 導致報表頁面無法正常載入

**解決方案**：
- 移除 `pipelineTooltipFormatter` 函數
- 使用簡單的 `<Tooltip contentStyle={chartTheme.tooltip} />` 替代
- 更新測試檔案移除已刪除函數的測試案例

- 整合測試：正常
